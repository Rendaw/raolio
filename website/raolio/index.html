Raolio is a media player that synchronizes playback with other connected Raolio media players. Raolio is intended for private media performances in office or long-distance situations where turning the volume up really high isn't a solution. SourceForums

Get Raolio

The latest version is 0, updated Oct. 25, 2013. Arch x64 Server GUI CLI Ubuntu x64 Server GUI CLI Windows x64 GUI

GUI Notes

Use host 0.0.0.0 (e.g. "0.0.0.0:20758") to start a server.

GUI Keyboard

alt aAdd files to playlist
alt leftPlay previous song
alt rightPlay next song
alt spacePause/play

GUI Chat

/addAdd files to playlist
/listList songs in playlist
/nextPlay next song
/backPlay previous song
/play (MIN:SEC)Resume song if paused, play from MIN:SEC if specified.
/stopPause playback.
/handle NAMEChange your name.
/quitQuit
