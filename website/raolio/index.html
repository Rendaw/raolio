Raolio is a media player that synchronizes playback with other connected Raolio media players. Raolio is intended for private media performances in office or long-distance situations where turning the volume up really high isn't a solution.

Get Raolio

SourceForums

Keyboard

alt aAdd files to playlist
alt leftPlay previous song
alt rightPlay next song
alt spacePause/play

Chat

/a(dd)Add files to playlist
/l(ist)
/ls		List songs in playlist
/n(ext)Play next song
/b(ack)Play previous song
/p(lay) (MIN:SEC)Resume song if paused, play from MIN:SEC if specified.
/s(top)
/pa(use)		Pause playback.
/q(uit)
/e(xit)		Quit
